# Observability & Receipts

Eve provides comprehensive observability through job receipts, event streams, and cost tracking.

## Job receipts

Every completed job produces a receipt with token usage, cost, duration, and outcome.

## Event streams

Subscribe to real-time event streams for job state changes, deploys, and system events.

## Cost tracking

Track spend per job, per agent, and per project with built-in cost accounting.
